# Personal website

## Requirements

* [Node.js](https://nodejs.org)

## Installation

```bash
npm install
```

## Development

```bash
# Terminal 1
npm run dev:site

# Terminal 2
npm run dev:css
```

## Production

```bash
# All in one
npm run build

# Individual parts
npm run build:site
npm run build:css
```

## Linting

```bash
# All in one
npm run lint

# Individual components
npm run lint:js
npm run lint:css
npm run lint:language
```
