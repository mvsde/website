# fynn.be

## Requirements

- [Node.js 24](https://nodejs.org/)

## Installation

```sh
npm install
```

## Development server

```sh
node --run dev
```

## Production build

```sh
node --run build
```

## Formatting

```sh
node --run format
```

## Linting

```sh
# All tasks
node --run lint

# Individual tasks
node --run lint:css
node --run lint:js
node --run lint:markdown
node --run lint:format
```
